Leetcode 错题记录

​ 仅以此篇记录我 Leetcode 做题过程中踩过的坑和实用Tips

1. 2711. 对角线上不同值的数量差
  • 矩阵对角线条数 m+n-1
2. 2116. 判断一个括号字符串是否有效
  • 根据左括号 ‘(‘ 和 右括号 ‘)’ 维护 c 的取值范围
  • 其实只需要维护 c_min 和 c_max 即可
  • 需要注意的是 c_min的重置 即 <0 时变回 1(其实是-1, 1, 3….) 这样做是为了去除掉那些不可行(c<0)的路径,防止造成干扰(如果不重置以后加加加又给加成0了)
  • 最后判断是否能取到0
3. 2712. 使所有字符相等的最小成本
  • 本质就是比较 s[i-1] 和 s[i]
  • 再取 min(i,n-i)
  • 因为反转操作只会改变 s[i−1] 与 s[i] 是否相等，不会改变其他相邻字符是否相等（相等的都反转还是相等，不相等的都反转还是不相等），所以每对相邻字符其实是互相独立的，我们只需要分别计算这些不相等的相邻字符的最小成本，即 min(i,n−i)，累加即为答案。
4. 2680. 最大或值
  • 贪心：把k全乘到某一个num上
  • 记录pre和left的技巧
  • 记得运算时要强转为(long long)num, 不然过不了
5. 368. 最大整除子集
  • dp 从前往后 dp的大致思路就是正着不行倒着来，就是干 最重要的是要有感觉，看到这题时知道要用dp去解决。
  • 先 sort
  • 计算 f[i] 表示该位置最大的整除子集长度
  • 记录转移
6. 416. 分割等和子集
  • 先看sum的奇偶性
  • 再二维动态dp(如果你看到这忘了怎么dp，说明你不太熟练，请你再去做一遍这个题目)
  • 不要深搜，会TE，别问，问就是试过
7. 1922. 统计好数字的数目

快速幂

8. 394. 字符串解码

  • 用一个vector 来记录每一层的字符串

  • 字母分为两类：一类被编码 一类未被编码 (通过left_c和is_encode实现)

  • 当前级字符串赋值时，需要清空上级字符串

9. leetcode 446周赛-3524. 求出数组X的值I
  • DP 刷表法, 是一种不太常见的DP类型
10. 连续子数组相关问题

  • 2444.统计界定子数组数目

  • 2845.统计趣味子数组数目

  • 2799.统计完全子数组数目

常规解法都是枚举右端点维护左端点, 作差。
